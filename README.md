# No, I'm Not a Human - Official Web Hub

**Nombre del Equipo:** Ghost Productions  
**URL del Proyecto:** https://ghostproductions.pythonanywhere.com/

## Descripción del Proyecto
"No, I'm Not a Human" es una plataforma web inmersiva diseñada para expandir la narrativa del videojuego de terror psicológico homónimo. El sitio funciona como una terminal de acceso de la agencia ficticia AFAE, permitiendo a los usuarios consultar expedientes clasificados de personajes, analizar objetos de supervivencia y reportar anomalías mediante un sistema dinámico. El proyecto integra una arquitectura híbrida que combina la agilidad de archivos JSON para contenido masivo con la robustez de una base de datos relacional para la gestión de usuarios y comunicaciones oficiales, garantizando una experiencia interactiva y coherente con la estética del juego.

## Lenguajes y Tecnologías Utilizadas
El desarrollo se basó en un ecosistema tecnológico moderno y escalable:
- **HTML5:** Estructura semántica avanzada para accesibilidad y SEO.
- **CSS3:** Diseño responsivo con estética "Cyberpunk/Neon", utilizando Flexbox, Grid y animaciones personalizadas.
- **JavaScript (JS):** Lógica del lado del cliente para validaciones de seguridad (Regex), manejo de estados de la interfaz y efectos visuales.
- **Python (Flask):** Motor de backend para la gestión de rutas, procesamiento de formularios y control de sesiones de usuario.
- **SQL (SQLite):** Sistema de gestión de base de datos relacional para el almacenamiento persistente de credenciales de agentes y registros de contacto.
